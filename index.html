Unity UI dev’s guide to learning Unreal 5 (v0.1)

There are more similarity then difference in my experience of learning Unreal 5 when I have years of experience in Unity, and I hope this premier inspires you to learn something new.

Where to start

https://dev.epicgames.com/community/learning/courses/l3p/unreal-engine-your-first-hour-in-umg/6o5/introduction

“Your First Hour in UMG” is the best beginner UMG(Unreal Motion Graphic) tutorial that I found. The tutorial comes with a project already setup, and it provides a good starting point to learn how UI is displayed in Unreal.
Unreal 4 and Unreal 5 has minor differences in UI, but not too many to make what you learn here hard to translate to Unreal 5. This tutorial gives you the confidence to tackle other Unreal UI learnings.

Key take-aways from this tutorial:

  • Learn how to make a UI widget and use the Level Blueprint to add the widget to viewport, so that you can make changes in the widget and see the changes you made in the game
